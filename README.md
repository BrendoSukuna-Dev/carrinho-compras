<h1>Carrinho de compras</h1>
<h2>🖥 Projeto Concluido ✅ </h2> 

## 📋 Sobre o projeto
<p>Este é um projeto de um sistema de Carrinho de Compras dinâmico, desenvolvido para praticar a integração entre estrutura HTML, estilização avançada com CSS e lógica de negócios com JavaScript.</p>
<p>O projeto permite que o usuário selecione produtos de uma lista, defina a quantidade desejada e visualize o valor total da compra em tempo real, simulando a experiência básica de um e-commerce.</p>

## ✦ Principais funcionalidades:

<p>➥ Adição Dinâmica: Recupera o nome do produto e o valor unitário a partir de um campo de seleção.</p>
<p>➥ Cálculo de Subtotal: Multiplica automaticamente a quantidade escolhida pelo valor unitário do produto.</p>
<p>➥ Totalizador Geral: Mantém e exibe a soma de todos os itens adicionados ao carrinho.</p>
<p>➥ Interface Reativa: Atualiza a lista de produtos visualmente no navegador utilizando manipulação de DOM.</p>
<p>➥ Limpeza de Dados: Função para resetar o carrinho e zerar o valor total.</p>
